# Tesla and GameStop Stock and Revenue Analysis

## Project Overview
This project analyzes the historical stock prices and revenue data of Tesla (TSLA) and GameStop (GME) using Python.

## Objectives
- Extract stock data using the yfinance library.
- Scrape revenue data from web sources using requests and BeautifulSoup.
- Clean and preprocess the extracted data.
- Visualize stock prices and revenue trends using graphs.

## Technologies Used
- Python
- Pandas
- yfinance
- Requests
- BeautifulSoup
- Plotly
- Jupyter Notebook

## Files
- `Tesla_GameStop_Stock_Revenue_Analysis.ipynb` – Main notebook containing all code, analysis, and visualizations.
- `README.md` – Project documentation.

## Results
The project compares Tesla and GameStop stock performance with their corresponding revenue trends and visualizes the relationship between market activity and company financial performance.
